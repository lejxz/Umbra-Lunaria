# Session Log 039: Dashboard Tweaks

**Date:** 2026-07-21
**Phase:** 1.3 - Polish

## Overview
Small quality of life and aesthetic tweaks to the dashboard based on visual review.

## Changes
- **Chart Axes**: Adjusted the X-axis label rotation on the Activity Timeline and Donation Analytics charts from `-30` to `-45` degrees to improve legibility and fit.
- **Empty States**: Added custom SVG icons to the empty states to make them feel more intentional and designed:
  - Added a `WarEmptyIcon` (crossed broken shield) to the Current War card.
  - Added an `ActivityEmptyIcon` (pulse line) to the Activity Timeline empty state.
  - Added a `DonationEmptyIcon` (box with arrow) to the Clan Donations empty state.

These changes make the empty states feel less barren and complete the premium glassmorphism aesthetic even when data isn't fully populated yet.
