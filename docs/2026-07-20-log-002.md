# Log 002 — Design Proposal & Theming

**Date:** 2026-07-20
**Time:** 13:28 PM (+08:00)

## Summary of Session
Created a standalone HTML/CSS design proposal to establish the visual aesthetic (colors, fonts, glassmorphism) based on the Umbra Lunaria logo.

## Work Completed
- Extracted a dark theme color palette from the reference image (Deep midnight blues, vivid glowing purples, soft lilac text).
- Drafted `umbra_lunaria_design.html` (in the scratch directory) as a playable static HTML file using Tailwind CSS to demonstrate the dashboard layout, glass-pane styling, and typography choices.

## Decisions Made
- Chose `Cinzel` (serif) for the primary logo/headers to capture the mystical, elegant feel of the moon image.
- Chose `Inter` (sans-serif) for the data-heavy UI elements for maximum readability.
- Implemented a "glassmorphism" aesthetic (`backdrop-filter: blur`) with subtle glowing borders to mimic the nebulous, glowing atmosphere of the reference art.
